# Projeto: Cálculos para Construção Civil

## Descrição do Projeto
Este projeto é um conjunto de funções em Python desenvolvido para auxiliar no planejamento de obras, realizando cálculos de área, volume e estimativa de materiais como tijolos, cimento, piso, rejunte e tinta, além de calcular custos logísticos como frete e verificar a capacidade de transporte. O objetivo é automatizar cálculos comuns na construção civil, tornando o processo mais rápido e preciso.

## Instruções de Execução
Para executar o projeto, é necessário ter o Python instalado. O código deve ser salvo em um arquivo `.py` e executado em uma IDE ou pelo terminal. As funções podem ser chamadas diretamente no código, passando os valores desejados como parâmetros, e os resultados podem ser exibidos com o uso de `print`.

## Descrição das Principais Funcionalidades
O sistema realiza cálculos essenciais para uma obra, como área de superfícies, quantidade de tijolos, volume de concreto e número de sacos de cimento. Também inclui funções para acabamento, como cálculo de caixas de piso, quantidade de rejunte e tinta necessária, além de recursos de logística que calculam o custo de frete e verificam se o peso de uma carga está dentro da capacidade de um caminhão.
